# Python Examples

Python examples for the Autofiller SDK.

## Setup

```bash
pip install autofiller-sdk
```

## Examples

### Basic Extraction

```bash
python 01_basic_extraction.py
```

### Async Extraction

```bash
python 02_async_extraction.py
```

### Batch Processing

```bash
python 03_batch_processing.py
```

## Environment Variables

```bash
export AUTOFILLER_API_KEY="your-api-key"
```
