# TypeScript Examples

TypeScript/Node.js examples for the Autofiller SDK.

## Setup

```bash
npm install
```

## Examples

### Basic Extraction

```bash
npm run example:basic
# or
npx tsx src/01-basic-extraction.ts
```

### Async Extraction

```bash
npm run example:async
# or
npx tsx src/02-async-extraction.ts
```

### Batch Processing

```bash
npm run example:batch
# or
npx tsx src/03-batch-processing.ts
```

## Environment Variables

```bash
export AUTOFILLER_API_KEY="your-api-key"
```
