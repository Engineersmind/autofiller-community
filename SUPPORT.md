## How to Ask Good Questions

Help us help you! When asking for help:

1. **Describe what you're trying to do**
2. **Share what you've tried**
3. **Include error messages** (full stack traces)
4. **Provide a minimal example** that reproduces the issue
5. **Mention your environment** (SDK version, Node/Python version, OS)

Example:

```markdown
## What I'm trying to do
Extract invoice data from a multi-page PDF using the Python SDK.

## What I've tried
```python
result = client.extract(file='./invoice.pdf', domain_pack='invoice-standard')
```

## Error I'm getting
```
ValidationError: Document exceeds maximum page limit (15 > 10)
```

## Environment
- autofiller-sdk 0.1.0
- Python 3.11
- macOS 14.0
```
